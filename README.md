📊 CGPA & SGPA Calculator (Android App for VTU Students)

A Kotlin-based Android app for VTU students to calculate CGPA and SGPA accurately from semester-wise marks.
The complete project is available in this repository with both UI design and functional logic.
Built in Android Studio, the app ensures secure, accurate, and reliable computations with a clean and intuitive UI.

🚀 Features

✅ Calculate both CGPA and SGPA as per VTU standards

✅ Convert CGPA into percentage instantly

✅ Organized and modular Kotlin codebase

✅ Clean XML-based UI layouts

✅ Additional custom designs using drawables

✅ Lightweight, accurate, and user-friendly

🛠️ Project Structure

📦 Installation & Setup

Clone the repository:

git clone https://github.com/<your-username>/CGPA-Calculator.git


Open in Android Studio

Go to File → Open, then select the project folder.

Allow Gradle sync to complete.

Build & Run

Connect an Android device or start an emulator.

Click ▶️ Run in Android Studio.
